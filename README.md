# crud-shared

## 📍 如何使用？

### 安装

```bash
npm i crud-shared
```

### 使用

```js
import { xxx } from 'crud-shared';
```

### 说明

- 如果想在自己项目中引用，可查看 [Demo](https://weeeisir.github.io/crud-shared/) 写法。
- 正逐步构想并添加，如果你有好的想法和建议，或者问题，欢迎 [issue](https://github.com/weeeiSir/crud-shared/issues)
